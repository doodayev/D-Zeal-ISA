# D-Zeal-ISA
This is a 64-bit CPU Instruction Set Architecture by the name of "D-Zeal". 64-bit registers, 64-bit memory addresses, 64-bit instructions. Also, every instruction has its instruction fields in multiples of 8 so you can read each instruction in hexadecimal format quite easily.
